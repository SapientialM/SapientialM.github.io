C++ STL

标准模板库 STL（Standard Template Library），C++ 标准库的一部分，只需要 #include 即可，是C++算法的神器。

STL 有六大部件：

  • 容器（Containers）
  • 分配器（Allocators）
  • 算法（Algorithm）
  • 迭代器（Iterators）
  • 适配器（Adapters）
  • 仿函数（Functors）

要真正提高 C++ 编程（刷题）效率，需要将STL六大部件结合使用，有些简单介绍也不重复，参考C++ STL 教程 | 菜鸟教程 (runoob.com) 就可以了，我在这里就是记录一些笔记，提供方便。

vector 动态数组

vector，动态数组，又称变长数组，可以随时进行添加和删除元素。

头文件：

初始化：

代码 作用
vector<T> v; 名为 v 的一维数组，元素类型为 T
vector<T> v(10); 数组初始大小为 10
vector<int> v(10,1); 数组元素默认初始值为 1
vector<int> v{1,2,3,4,5}; 指定初始化数组元素
vector<int> a{1,2,3,4}; vector<int> b(a); 拷贝初始化
vector<T> v[10]; 二维数组，实际上是存储vector元素的固定数组。可以理解为固定行数10的二维数组
vector<vector<T>> v ; 二维数组，行列均可变，实际上是存储vector元素的变长数组

访问：（后面有关于迭代器的说明）

方法 使用
下标法 同普通数组，如 v[0]
迭代器 vector<int>::iterator it = v.begin(); cout << *it;
Range-Based-For（C++11） for (auto &&i : v) { cout << i;}

方法函数：（T 即 template<typename T>模板）

方法 作用
v.front() 返回第一个元素，$O(1)$
v.back() 返回最后一个元素，$O(1)$
v.push_back(element) 尾部添加元素，$O(1)$
v.pop_back() 尾部弹出元素，$O(1)$
v.size() 返回数组元素个数，$O(1)$
v.clear() 清空数组，$O(N)$
v.resize(n,v) 设置数组大小为 n，且元素默认值为 v，无v则默认0，$O(N)$
v.insert(it, element) 在迭代器 it 处插入元素 element，$O(N)$
v.erase(start, end) 删除[start, end) 区间内的所有元素，$O(N)$
v.begin() 返回首元素的迭代器，$O(1)$
v.end() 返回尾元素的迭代器，$O(1)$
v.empty() 判断数组是否为空，为空则true，$O(1)$
